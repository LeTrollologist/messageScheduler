# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this plugin, please do not report it publicly. Instead, please open a private issue if the platform supports it, or contact the maintainers directly.

## Safety Principles
- **Local Data**: All scheduled messages are stored locally on your machine within Vencord's private settings store.
- **No External Tracking**: This plugin does not communicate with any external servers except for Discord's own internal API for sending messages.
- **Minimal Permissions**: The plugin only requests the permissions necessary to read your input and send messages on your behalf at the scheduled time.

## Precautions
As with any client modification, ensure you are building the plugin from source and only downloading Vencord from official sources to protect your account credentials and personal data.
